fiesta.util.fb_func_stats
=========================
Given a `Fixed Budget (FB) method <./fixed_budget.html>`_ like 
:func:`~fiesta.fiesta.sequential_halving` it will run that 
function with the given keyword arguments *N* times and will report the 
the probability that the FB function chose the best model from all *N* runs.
